\begin{abstract}
RCC Nexus v1.7 is an additive adoption-profile and utility-evidence layer above
RCC-N v1.6. RCC tells an agent what the repository means. RCC-N tells the agent
where it is. Validation tells the agent whether reality agreed. RCC-N v1.0 made
repository context geometric. RCC-N v1.1 made geometry measurable. RCC-N v1.2
made enforcement outcomes repair geometry. RCC-N v1.3 made repair history
diagnostic. RCC-N v1.4 made diagnostic patterns federated. RCC-N v1.5 made
federated evidence admissible only under comparable context. RCC-N v1.6 made
the standard itself governable. RCC-N v1.7 makes RCC-N adoptable: repositories
select the lightest sufficient profile according to risk, complexity,
validation need, and federation need. Minimal viable RCC-N is defined for
low-overhead adoption; higher profiles activate stronger rule sets, ledgers,
diagnosis, federation, and critical-safety gates. RCC-N v1.7 also requires
evidence-of-utility measurement before claiming maintenance improvement and
adds corrective version governance so standard review may recommend profile
updates, compatibility bridges, rule deprecations, feature downgrades, claim
tightening, or future version candidates without deleting history or pretending
correction is failure.
\end{abstract}

\begin{longtable}{p{0.16\textwidth}p{0.74\textwidth}}

\midrule

Small or low-risk repositories that need AI orientation and basic validation

evidence discipline. \\

\section{Profile Selection Function}

\begin{definition}[Governance Need]
Governance need is the estimated level of RCC-N governance required by a
repository based on risk, complexity, validation burden, claim surface, and
federation need.
\end{definition}

\[
\boxed{
G_{need}
=
w_rR
+
w_cC
+
w_vV
+
w_fF
+
w_pP.
}
\]

where:

\[
R=\text{risk of context error},
\]

\[
C=\text{repository complexity},
\]

\[
V=\text{validation burden},
\]

\[
F=\text{federation need},
\]

\[
P=\text{public claim surface}.
\]

Default weights:

\[
(w_r,w_c,w_v,w_f,w_p)
=
(0.25,0.20,0.20,0.20,0.15).
\]

Profile bands:

\[
0.00\leq G_{need}<0.25
\Rightarrow Lite.
\]

\[
0.25\leq G_{need}<0.50
\Rightarrow Standard.
\]

\[
0.50\leq G_{need}<0.70
\Rightarrow Full.
\]

\[
F\geq 0.70
\Rightarrow Federated.
\]

\[
R\geq 0.80
\vee
V\geq 0.80
\vee
P\geq 0.80
\Rightarrow CriticalReview.
\]

\begin{rulebox}[Lightest Sufficient Profile Rule]
A repository should use the lightest profile whose governance capacity meets or
exceeds governance need.
\end{rulebox}

\[
Profile(repo)
=
\min

\section{Reference Runtime Pseudocode v1.7}
%──────────────────────────────────────────────────────────────────────────────

This pseudocode adds adoption-profile checks, minimal viable RCC-N checks,
profile-gated rule activation, utility evidence checks, governance-cost checks,
computable SUP scaffolding, and corrective governance checks.

\begin{lstlisting}[language=Python]
from dataclasses import dataclass, asdict
from pathlib import Path
from typing import Dict, List, Any, Optional
import json


CHECKER_MATURITY = "Q2-profile-utility-skeleton"

PROFILES = ["Lite", "Standard", "Full", "Federated", "Critical"]

PROFILE_RANK = {
    "Lite": 1,
    "Standard": 2,
    "Full": 3,
    "Federated": 4,
    "Critical": 5,
}


@dataclass
class Finding:
    code: str
    message: str
    severity: str
    tier: str
    path: Optional[str] = None


def load_json(path: Path) -> Dict[str, Any]:
    if not path.exists():
        return {}
    return json.loads(path.read_text(encoding="utf-8"))


def clamp01(value: float) -> float:
    try:
        return max(0.0, min(1.0, float(value)))
    except Exception:
        return 0.0


def governance_need(profile_selection: Dict[str, Any]) -> float:
    weights = {
        "risk": 0.25,
        "complexity": 0.20,
        "validation_need": 0.20,
        "federation_need": 0.20,
        "public_claim_surface": 0.15,
    }
    return clamp01(sum(
        weights[k] * clamp01(profile_selection.get(k, 0.0))
        for k in weights
    ))


def recommended_profile(score: float, profile_selection: Dict[str, Any]) -> str:
    risk = clamp01(profile_selection.get("risk", 0.0))
    validation = clamp01(profile_selection.get("validation_need", 0.0))
    federation = clamp01(profile_selection.get("federation_need", 0.0))
    claims = clamp01(profile_selection.get("public_claim_surface", 0.0))

    if risk >= 0.80 or validation >= 0.80 or claims >= 0.80:
        return "Critical"
    if federation >= 0.70:
        return "Federated"
    if score >= 0.50:
        return "Full"
    if score >= 0.25:
        return "Standard"
    return "Lite"


def check_profile_selection(index: Dict[str, Any]) -> List[Finding]:
    findings: List[Finding] = []
    repo = index.get("repository", {})
    declared = repo.get("adoption_profile")

    if declared not in PROFILES:
        findings.append(Finding(
            "RCCN093",
            f"Missing or invalid adoption profile: {declared}",
            "error",
            "profile",
        ))
        return findings

    selection = index.get("profile_selection", {})
    score = governance_need(selection)
    recommended = recommended_profile(score, selection)

    if PROFILE_RANK[declared] > PROFILE_RANK[recommended] + 1:
        findings.append(Finding(
            "RCCN094",
            f"Repository may be over-governed: declared {declared}, recommended {recommended}.",
            "warning",
            "profile",
        ))

    if PROFILE_RANK[declared] < PROFILE_RANK[recommended]:
        findings.append(Finding(
            "RCCN095",
            f"Repository may be under-governed: declared {declared}, recommended {recommended}.",
            "error",
            "profile",
        ))

    return findings


def check_minimal_viable_rccn(index: Dict[str, Any]) -> List[Finding]:
    findings: List[Finding] = []
    minimal = index.get("minimal_viable_rccn", {})

    required = [
        "public_spine_present",
        "readme_trisection_present",
        "ai_agent_section_present",
        "major_echo_locations_present",
        "validation_commands_declared",
        "non_claim_locks_present",
        "done_criteria_present",
    ]

    missing = [k for k in required if not minimal.get(k)]
    if missing:
        findings.append(Finding(
            "RCCN096",
            f"Minimal viable RCC-N missing required items: {missing}",
            "error",
            "minimal",
        ))

    return findings


def check_profile_gated_rules(index: Dict[str, Any], registry: Dict[str, Any]) -> List[Finding]:
    findings: List[Finding] = []

    declared_profile = index.get("repository", {}).get("adoption_profile", "Lite")
    declared_rank = PROFILE_RANK.get(declared_profile, 1)

    active_claim_rules = set(index.get("profile_gated_rules", {}).get("claim_required_rules", []))

    for rule in registry.get("rules", []):
        code = rule.get("code")
        required_profile = rule.get("required_profile", "Lite")
        required_rank = PROFILE_RANK.get(required_profile, 1)

        if required_rank > declared_rank and code not in active_claim_rules:
            if rule.get("reported_missing"):
                findings.append(Finding(
                    "RCCN097",
                    f"Rule reported missing but inactive under selected profile: {code}",
                    "warning",
                    "profile",
                ))

    return findings


def check_utility_evidence(index: Dict[str, Any]) -> List[Finding]:
    findings: List[Finding] = []
    utility = index.get("utility_evidence", {})

    if not utility.get("claims_rccn_improves_maintenance"):
        return findings

    if not utility.get("baseline_comparison_present"):
        findings.append(Finding(
            "RCCN098",
            "RCC-N utility improvement claimed without baseline comparison.",
            "error",
            "utility",
        ))

    if not utility.get("task_trace_present") or not utility.get("validation_outcome_present"):
        findings.append(Finding(
            "RCCN099",
            "RCC-N utility improvement claimed without task trace or validation outcome.",
            "error",
            "utility",
        ))

    if not utility.get("cost_accounting_present"):
        findings.append(Finding(
            "RCCN100",
            "RCC-N utility improvement claimed without governance cost accounting.",
            "warning",
            "cost",
        ))

    return findings


def compute_artifact_sup(index: Dict[str, Any], registry: Dict[str, Any], coverage: Dict[str, Any]) -> Dict[str, Any]:
    rules = registry.get("rules", [])
    coverage_rows = coverage.get("coverage", [])

    coverage_map = {
        row.get("rule"): row.get("status")
        for row in coverage_rows
        if row.get("rule")
    }

    high_severity = [
        r for r in rules
        if r.get("default_severity") in {"blocking", "error"}
    ]

    high_gap = [
        r for r in high_severity
        if coverage_map.get(r.get("code")) in {None, "manual", "partial", "unimplemented"}
    ]

    rule_pressure = len(high_gap) / max(1, len(high_severity))

    manual_high = [
        r for r in high_severity
        if coverage_map.get(r.get("code")) == "manual"
    ]

    manual_pressure = len(manual_high) / max(1, len(high_severity))

    signals = index.get("standard_update_signals", {})

    components = {
        "rule_pressure": clamp01(rule_pressure),
        "schema_pressure": clamp01(signals.get("schema_pressure", 0.0)),
        "profile_pressure": clamp01(signals.get("profile_pressure", 0.0)),
        "compatibility_pressure": clamp01(signals.get("compatibility_pressure", 0.0)),
        "overclaim_pressure": clamp01(signals.get("overclaim_pressure", 0.0)),
        "manual_pressure": clamp01(manual_pressure),
    }

    weights = index.get("standard_update_pressure", {}).get("weights", {
        "rule_pressure": 0.20,
        "schema_pressure": 0.20,
        "profile_pressure": 0.15,
        "compatibility_pressure": 0.15,
        "overclaim_pressure": 0.20,
        "manual_pressure": 0.10,
    })

    score = sum(float(weights.get(k, 0.0)) * v for k, v in components.items())
    threshold = float(index.get("standard_update_pressure", {}).get("threshold", 0.70))

    return {
        "score": clamp01(score),
        "threshold": threshold,
        "components": components,
        "computed_components": ["rule_pressure", "manual_pressure"],
        "manual_components": [
            k for k in components
            if k not in {"rule_pressure", "manual_pressure"}
        ],
        "version_review_required": score >= threshold,
    }


def check_sup_disclosure(index: Dict[str, Any], sup: Dict[str, Any]) -> List[Finding]:
    findings: List[Finding] = []
    disclosed = set(index.get("computed_sup", {}).get("manual_inputs_disclosed", []))

    for component in sup.get("manual_components", []):
        if component not in disclosed:
            findings.append(Finding(
                "RCCN101",
                f"SUP component manually supplied without disclosure: {component}",
                "warning",
                "sup",
            ))

    return findings


def check_corrective_governance(index: Dict[str, Any]) -> List[Finding]:
    findings: List[Finding] = []

    for correction in index.get("standard_corrections", []):
        if not correction.get("history_preserved"):
            findings.append(Finding(
                "RCCN102",
                "Standard correction/downgrade/deprecation performed without preserving history.",
                "error",
                "governance",
            ))

        if correction.get("type") == "rule_deprecation":
            if not correction.get("replacement_rule") and not correction.get("no_replacement_reason"):
                findings.append(Finding(
                    "RCCN103",
                    "Rule deprecated without replacement rule or no-replacement reason.",
                    "warning",
                    "governance",
                ))

        if correction.get("type") == "profile_retraction":
            required = ["scope", "affected_repo_class", "migration_path"]
            missing = [k for k in required if not correction.get(k)]
            if missing:
                findings.append(Finding(
                    "RCCN104",
                    f"Profile retraction missing fields: {missing}",
                    "error",
                    "governance",
                ))

    return findings


def check_gen_boundary(index: Dict[str, Any]) -> List[Finding]:
    findings: List[Finding] = []
    gen = index.get("gen_boundary", {})

        findings.append(Finding(

            "RCCN106",
            "GEN boundary non-claim lock missing.",
            "blocking",
            "governance",
        ))

    return findings

Thus RCC-N v1.7 evolves RCC-N from a complete high-trust repository-navigation
standard into an adoptable, profile-gated, empirically measurable governance
system. RCC-N v1.6 made the standard governable through rule registry, checker
coverage, misuse detection, and version-review triggers. RCC-N v1.7 makes that
standard practical: repositories choose Lite, Standard, Full, Federated, or
Critical profiles based on risk and need; Minimal Viable RCC-N provides a low
overhead entry point; utility claims require baseline comparison and cost
accounting; SUP components become computable from artifacts where possible; and
standard correction, downgrade, deprecation, and profile retraction become
governed actions that preserve history. The result is not proof of correctness,
intelligence, security, or safety. The result is an adoptable repository
navigation standard whose governance can be scaled, measured, corrected, and
eventually abstracted into GEN — Governed Epistemic Navigation.
