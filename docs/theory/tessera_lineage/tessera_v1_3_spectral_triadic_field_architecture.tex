\begin{abstract}
TESSERA v1.3 is an additive research-integrated evolution of the prior TESSERA
line. Earlier simulations exposed three linked facts: a sparse damped field can
remain highly stable; a raw coherence formula can collapse in quiet fields; and
triadic warning/block/memory separation can reduce harmful gate fusion relative
to monolithic policies. The research synthesis adds stronger foundations from
geometric deep learning, sparse training, graph spectra, bounded recurrent
dynamics, neural-field theory, dendritic and neuromodulatory gating, synaptic
consolidation, and homeostatic plasticity. TESSERA v1.3 therefore upgrades the
architecture from symbolic geometry to spectral geometry: every routing field
must declare its graph, normalized operator, damping policy, and stability
margin. It also upgrades the triad from a threshold stack to an asymmetric
control law: warning optimizes sensitivity, block optimizes constraint, and
memory optimizes precision under certification. The core claim remains bounded:
TESSERA v1.3 is a falsifiable sparse-field architecture for calibrated
selective preservation, not proof of consciousness, AGI, universal phi damping,
or guaranteed superiority over dense networks.
\end{abstract}

\section{Spectral Routing Law}
%----------------------------------------------------------------------------

\begin{definition}[Sparse Field Graph]
A TESSERA field is a sparse graph dynamical system
\[
G=(V,E)
\]
with state
\[
F_t\in\mathbb{R}^{N},\quad N=|V|.
\]
\end{definition}

Let \(P\) be the normalized routing operator associated with \(G\). A canonical
field update is:

\[
\boxed{
F_{t+1}=\sigma(\alpha P F_t + B u_t)
}
\]

where:

\[
0<\alpha<1,
\]

\[
\sigma=\text{bounded 1-Lipschitz activation},
\]

\[
B=\text{input projection operator},
\]

\[
u_t=\text{input or telemetry vector at time }t.
\]

\begin{rulebox}[Spectral Declaration Rule]
A valid v1.3 implementation must declare:
\[
(G,P,\rho(P),\alpha,\alpha\rho(P),\operatorname{normPolicy}).
\]
If the spectral radius or normalization policy is unknown, the implementation
must be marked exploratory and may not claim stability beyond observation.
\end{rulebox}

\begin{proposition}[Contractive Field Condition]
If \(\sigma\) is 1-Lipschitz, \(\|P\|\leq1\), and \(0<\alpha<1\), then in the
zero-input case the update map is contractive:
\[
\|F_{t+1}-\tilde{F}_{t+1}\|
\leq
\alpha\|F_t-\tilde{F}_t\|.
\]
\end{proposition}

\begin{remark}
This explains why low Delta-Phi can be easy to obtain in a damped field. Low
Delta-Phi is not itself proof of useful memory; it may simply mean the operator
is strongly contractive.
\end{remark}

\section{Calibration Manifold}
%----------------------------------------------------------------------------

v1.3 replaces a single coherence score with a calibrated metric manifold.

Define amplitude:

\[
A_t=\frac{1}{N}\sum_i |F_t(i)|.
\]

Define energy:

\[
E_t=\frac{1}{N}\sum_i F_t(i)^2.
\]

Define field variance:

\[
V_t=\operatorname{Var}(F_t).
\]

Define residual:

\[
\Delta\Phi_t=\frac{1}{N}\sum_i |F_t(i)-F_{t-1}(i)|.
\]

Two residual stability channels are logged:

\[
\Omega^{raw}_t=\frac{1}{1+\beta|\Delta\Phi_t|},
\]

\[
\Omega^{exp}_t=e^{-\beta|\Delta\Phi_t|}.
\]

\begin{definition}[Calibration Window]
A calibration window is a bounded origin segment:
\[
W_0=\{F_1,\ldots,F_m\}
\]
used to estimate robust amplitude, residual, variance, and coherence ranges.
\end{definition}

Robust estimates:

\[
A^*=\operatorname{median}(A_W),
\]

\[
s_A=\operatorname{MAD}(A_W).
\]

Amplitude band:

\[
A_{min}=Q_{0.10}(A_W),
\quad
A_{max}=Q_{0.90}(A_W).
\]

\begin{rulebox}[Stale Calibration Rule]
If the input distribution, topology, damping, projection gain, or validation
context changes beyond the declared TTL, calibration becomes stale and memory
updates are blocked until recalibrated.
\end{rulebox}

+

\begin{lstlisting}[language=Python]
def tessera_v13_step(field, input_u, config, calibration, gates):
    # 1. Spectral routing declaration
    P = config.normalized_operator
    alpha = config.damping_alpha
    assert alpha * config.spectral_radius < config.stability_margin_limit

    # 2. Field update
    projected = config.B @ input_u
    next_field = bounded_activation(alpha * (P @ field) + projected)

    # 3. Raw metrics
    raw = compute_raw_metrics(field, next_field)

    # 4. Calibrated metrics
    cal = compute_calibrated_metrics(raw, calibration)

    # 5. Triadic gates
    warning = warning_gate(raw, cal, gates.warning_policy)
    blocked = block_gate(raw, cal, gates.block_policy)
    memory = memory_gate(
        raw=raw,
        cal=cal,
        blocked=blocked,
        rhp_pass=config.rhp_pass,
        evidence_pass=config.evidence_pass,
        validation_pass=config.validation_pass,
        certificate_ready=config.certificate_ready,
    )

    # 6. Action ceiling
    if memory:
        action = "bounded_certified_update"
    elif blocked:
        action = "wound_or_quarantine"
    elif warning:
        action = "bounded_output_with_warning"
    else:
        action = "bounded_output"

    # 7. Ledger
    cert = make_v13_certificate(raw, cal, warning, blocked, memory, action)
    return next_field, cert
\end{lstlisting}
