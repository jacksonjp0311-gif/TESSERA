# TESSERA Alignment and Geometry Gap

## Gap

Tessera had working loop pieces, but the public geometry was not aligned.

```text
README said one thing.
Loopbook said another.
PowerShell carried too much logic.
Bash was too thin.
Lessons existed but were not visibly charted.
Observer CLI existed but was not guaranteed as the first visible surface.
Engine patch notes leaked into the repository root.
```

## Repair Law

```text
No command without registry.
No loop without Observer-first launch.
No feature without Loopbook/lessons/chart alignment.
No root-level engine patch notes.
No PowerShell-owned loop logic.
```
