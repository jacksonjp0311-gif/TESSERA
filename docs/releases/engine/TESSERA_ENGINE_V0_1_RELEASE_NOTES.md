# TESSERA Engine v0.1 Release Notes

## Identity

**Name:** Tessera  
**Version:** Engine v0.1  
**Purpose:** First runnable sparse spectral compressive memory network.

## What was built

- Python package and CLI: `tessera`
- Sparse graph operators: Q4, ring, random sparse, dense
- PyTorch TESSERANet with spectral field, encoder, decoder, predictor
- Synthetic telemetry generator
- Chronological split discipline
- Dataset manifest and leakage guard
- Rate-distortion metrics
- Triadic warning/block/memory gates
- Replay-promoted memory scoring
- Wound ledger
- Baselines: persistence, PCA codec, random projection, EWMA
- Evidence package and transfer certificate
- Plots and run report
- Unit tests
- All-One PowerShell runner

## Claim boundary

This is a reference engine. It may demonstrate that the architecture runs and emits evidence on synthetic telemetry. It does not prove real telemetry transfer, safety, AGI, consciousness, or superiority over baselines.

## Next version

Tessera Engine v0.2 should add real telemetry dataset adapters and the V1.7 transfer protocol end-to-end.
